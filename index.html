What It Does

Everything You Need to Parse Resumes

A full-stack applicant tracking system with an NLP extraction engine and a premium dark-mode UI.

NLP Extraction Engine

Uses spaCy's en_core_web_sm model for named entity recognition to identify candidate names and context-aware data.

PDF & DOCX Support

Handles both PDF (via pdfplumber) and DOCX (via python-docx) formats with intelligent text normalization.

Persistent Storage

All parsed candidates are saved to a SQLite database via SQLAlchemy ORM with full CRUD support.

Live Search & Filter

Real-time candidate search and skill-based filtering on the dashboard — no page reload needed.

REST API

JSON endpoints for listing, exporting, and deleting candidates — making integration with other tools simple.

Glassmorphism UI

A premium dark-mode interface with animated glows, glassmorphism cards, hover animations, and tab-based detail views.
